# Vendored OpenTelemetry `.proto` files

Source of truth for the C# types under `OpenTelemetry.Proto.*` generated by
`Grpc.Tools` in `src/Homespun.Server/Homespun.Server.csproj`.

## Regeneration discipline

1. Pick a tag from <https://github.com/open-telemetry/opentelemetry-proto/tags>.
2. Overwrite every file under `opentelemetry/proto/**/*.proto` with the
   verbatim contents at that tag.
3. Update `PROTO_VERSION` to name the new tag + commit SHA.
4. Run `dotnet build` — the generated types live in `obj/` and recompile on
   proto change.
5. Bump the tag in one focused PR, never mixed with receiver logic, so any
   wire-format regression can be bisected cleanly.
